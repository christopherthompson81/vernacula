# L-Esportazzjoni tar-Riżultati jew Traskrizzjonijiet

Tista' tesporta traskrizzjoni lesta f'diversi formati ta' fajl biex tużaha f'applikazzjonijiet oħra.

## Kif Tesporta

1. Iftaħ xogħol lest. Ara [Il-Fetħ ta' Xogħlijiet Lesti](loading_completed_jobs.md).
2. Fil-veduta **Results**, ikklikkja `Export Transcript`.
3. Id-djalogu **Export Transcript** jinfetaħ. Agħżel format mill-lista **Format** drop-down.
4. Ikklikkja `Save`. Jinfetaħ djalogu biex issalva l-fajl.
5. Agħżel folder ta' destinazzjoni u isem tal-fajl, imbagħad ikklikkja **Save**.

Messaġġ ta' konferma jidher fil-qiegħ tad-djalogu juri l-perkors sħiħ tal-fajl salvat.

## Formati Disponibbli

| Format | Estensjoni | L-aħjar għal |
|---|---|---|
| Excel | `.xlsx` | Analiżi ta' spreadsheet b'kolonni għall-kelliem, timestamps, u kontenut. |
| CSV | `.csv` | Importa f'kwalunkwe spreadsheet jew għodda tad-dejta. |
| JSON | `.json` | Ipproċessar programmatiku. |
| SRT Subtitles | `.srt` | Tagħbija f'editers tal-vidjo jew plejers tal-midja bħala sottotitoli. |
| Markdown | `.md` | Dokumenti ta' test sempliċi li jinqraw faċilment. |
| Word Document | `.docx` | Kondiviżjoni ma' utenti ta' Microsoft Word. |
| SQLite Database | `.db` | Esportazzjoni sħiħa tad-database għal queries personalizzati. |

## Ismijiet tal-Kelliema fl-Esportazzjonijiet

Jekk assenjajt ismijiet murija lill-kelliema, dawk l-ismijiet jintużaw fil-formati kollha ta' esportazzjoni. Biex taġġorna l-ismijiet qabel l-esportazzjoni, ikklikkja `Edit Speaker Names` l-ewwel. Ara [L-Editjar tal-Ismijiet tal-Kelliema](editing_speaker_names.md).
