# Редактирование транскриптов

**Редактор транскриптов** позволяет просматривать результаты ASR, исправлять текст, переименовывать спикеров прямо в редакторе, корректировать временны́е метки сегментов и отмечать сегменты как проверенные — всё это во время прослушивания исходного аудио.

## Открытие редактора

1. Загрузите завершённое задание (см. [Загрузка завершённых заданий](loading_completed_jobs.md)).
2. В представлении **Результаты** нажмите `Edit Transcript`.

Редактор откроется в отдельном окне и может оставаться открытым рядом с основным приложением.

## Интерфейс

Каждый сегмент отображается в виде карточки с двумя расположенными рядом панелями:

- **Левая панель** — исходный вывод ASR с цветовой индикацией уверенности для каждого слова. Слова, в которых модель была менее уверена, выделяются красным; слова с высокой уверенностью отображаются обычным цветом.
- **Правая панель** — редактируемое текстовое поле. Вносите исправления здесь; по мере ввода подсвечиваются отличия от исходного текста.

Над каждой карточкой отображаются метка спикера и временной диапазон. Нажмите на карточку, чтобы выделить её и показать значки действий. Наведите курсор на любой значок, чтобы увидеть всплывающую подсказку с описанием его функции.

## Описание значков

### Панель воспроизведения

| Значок | Действие |
|--------|----------|
| ▶ | Воспроизвести |
| ⏸ | Пауза |
| ⏮ | Перейти к предыдущему сегменту |
| ⏭ | Перейти к следующему сегменту |

### Действия на карточке сегмента

| Значок | Действие |
|--------|----------|
| <mdl2 ch="E77B"/> | Переназначить сегмент другому спикеру |
| <mdl2 ch="E916"/> | Изменить время начала и окончания сегмента |
| <mdl2 ch="EA39"/> | Скрыть или отобразить сегмент |
| <mdl2 ch="E72B"/> | Объединить с предыдущим сегментом |
| <mdl2 ch="E72A"/> | Объединить со следующим сегментом |
| <mdl2 ch="E8C6"/> | Разделить сегмент |
| <mdl2 ch="E72C"/> | Повторно выполнить ASR для этого сегмента |

## Воспроизведение аудио

В верхней части окна редактора расположена панель воспроизведения:

| Элемент управления | Действие |
|--------------------|----------|
| Значок «Воспроизвести / Пауза» | Запустить или приостановить воспроизведение |
| Полоса перемотки | Перетащите, чтобы перейти к любому месту в аудио |
| Ползунок скорости | Изменить скорость воспроизведения (0.5× – 2×) |
| Значки «Пред. / След.» | Перейти к предыдущему или следующему сегменту |
| Выпадающий список режимов воспроизведения | Выбрать один из трёх режимов воспроизведения (см. ниже) |
| Ползунок громкости | Изменить уровень громкости |

Во время воспроизведения текущее произносимое слово подсвечивается в левой панели. При паузе после перемотки подсветка перемещается к слову в точке перемотки.

### Режимы воспроизведения

| Режим | Поведение |
|-------|-----------|
| `Single` | Воспроизвести текущий сегмент один раз, затем остановиться. |
| `Auto-advance` | Воспроизвести текущий сегмент; по завершении отметить его как проверенный и перейти к следующему. |
| `Continuous` | Воспроизводить все сегменты последовательно, не отмечая ни один как проверенный. |

Выберите активный режим из выпадающего списка на панели воспроизведения.

## Редактирование сегмента

1. Нажмите на карточку, чтобы выделить её.
2. Отредактируйте текст в правой панели. Изменения сохраняются автоматически при переходе к другой карточке.

## Переименование спикера

Нажмите на метку спикера в выделенной карточке и введите новое имя. Нажмите `Enter` или щёлкните в другом месте для сохранения. Новое имя применяется только к этой карточке; чтобы переименовать спикера глобально, воспользуйтесь функцией [Редактирование имён спикеров](editing_speaker_names.md) в представлении «Результаты».

## Верификация сегмента

Нажмите флажок `Verified` на выделенной карточке, чтобы отметить сегмент как проверенный. Статус верификации сохраняется в базе данных и отображается в редакторе при последующих загрузках.

## Скрытие сегмента

Нажмите `Suppress` на выделенной карточке, чтобы исключить сегмент из экспорта (полезно для шума, музыки или других неречевых фрагментов). Нажмите `Unsuppress`, чтобы восстановить его.

## Изменение временных меток сегмента

Нажмите `Adjust Times` на выделенной карточке, чтобы открыть диалог изменения временных меток. Прокрутите колёсико мыши над полем **Start** или **End**, чтобы изменять значение с шагом 0,1 секунды, или введите значение вручную. Нажмите `Save` для применения изменений.

## Объединение сегментов

- Нажмите `⟵ Merge`, чтобы объединить выделенный сегмент с предшествующим ему сегментом.
- Нажмите `Merge ⟶`, чтобы объединить выделенный сегмент со следующим за ним сегментом.

Текст и временной диапазон обеих карточек объединяются. Это удобно, когда одна произнесённая фраза была разбита на два сегмента.

## Разделение сегмента

Нажмите `Split…` на выделенной карточке, чтобы открыть диалог разделения. Укажите точку разделения в тексте и подтвердите действие. Будут созданы два новых сегмента, охватывающих исходный временной диапазон. Это удобно, когда два отдельных высказывания были объединены в один сегмент.

## Повторный запуск ASR

Нажмите `Redo ASR` на выделенной карточке, чтобы повторно выполнить распознавание речи для аудио этого сегмента. Модель обрабатывает только фрагмент аудио данного сегмента и формирует новый транскрипт из единственного источника.

Используйте эту функцию, когда:

- Сегмент был получен в результате объединения и не может быть разделён (объединённые сегменты охватывают несколько источников ASR; повторный запуск ASR сворачивает их в один, после чего становится доступна функция `Split…`).
- Исходная транскрипция неудовлетворительного качества и вы хотите получить чистый повторный результат без ручного редактирования.

**Примечание:** Любой текст, уже введённый вами в правой панели, будет удалён и заменён новым результатом ASR. Для выполнения операции необходимо, чтобы аудиофайл был загружен; кнопка недоступна, если аудио недоступно.
