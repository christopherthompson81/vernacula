# Exportar resultados o transcripciones

Puede exportar una transcripción completada a varios formatos de archivo para utilizarla en otras aplicaciones.

## Cómo exportar

1. Abra un trabajo completado. Consulte [Cargar trabajos completados](loading_completed_jobs.md).
2. En la vista **Resultados**, haga clic en `Export Transcript`.
3. Se abre el cuadro de diálogo **Export Transcript**. Elija un formato en el menú desplegable **Format**.
4. Haga clic en `Save`. Se abre un cuadro de diálogo para guardar el archivo.
5. Elija una carpeta de destino y un nombre de archivo, luego haga clic en **Save**.

Aparece un mensaje de confirmación en la parte inferior del cuadro de diálogo con la ruta completa del archivo guardado.

## Formatos disponibles

| Formato | Extensión | Ideal para |
|---|---|---|
| Excel | `.xlsx` | Análisis en hojas de cálculo con columnas para el hablante, las marcas de tiempo y el contenido. |
| CSV | `.csv` | Importación en cualquier hoja de cálculo o herramienta de datos. |
| JSON | `.json` | Procesamiento mediante programación. |
| Subtítulos SRT | `.srt` | Carga en editores de vídeo o reproductores multimedia como subtítulos. |
| Markdown | `.md` | Documentos de texto sin formato legibles. |
| Documento Word | `.docx` | Compartir con usuarios de Microsoft Word. |
| Base de datos SQLite | `.db` | Exportación completa de la base de datos para consultas personalizadas. |

## Nombres de hablantes en las exportaciones

Si ha asignado nombres para mostrar a los hablantes, esos nombres se utilizan en todos los formatos de exportación. Para actualizar los nombres antes de exportar, haga clic primero en `Edit Speaker Names`. Consulte [Editar nombres de hablantes](editing_speaker_names.md).
