# Exportarea Rezultatelor sau Transcrierilor

Puteți exporta o transcriere finalizată în mai multe formate de fișier pentru a o utiliza în alte aplicații.

## Cum să Exportați

1. Deschideți o lucrare finalizată. Consultați [Încărcarea Lucrărilor Finalizate](loading_completed_jobs.md).
2. În vizualizarea **Rezultate**, faceți clic pe `Export Transcript`.
3. Se deschide dialogul **Export Transcript**. Alegeți un format din lista derulantă **Format**.
4. Faceți clic pe `Save`. Se deschide un dialog de salvare a fișierului.
5. Alegeți un folder de destinație și un nume de fișier, apoi faceți clic pe **Save**.

Un mesaj de confirmare apare în partea de jos a dialogului, afișând calea completă a fișierului salvat.

## Formate Disponibile

| Format | Extensie | Recomandat pentru |
|---|---|---|
| Excel | `.xlsx` | Analiză în foi de calcul, cu coloane pentru vorbitor, marcaje de timp și conținut. |
| CSV | `.csv` | Import în orice foaie de calcul sau instrument de date. |
| JSON | `.json` | Procesare programatică. |
| SRT Subtitles | `.srt` | Încărcare în editoare video sau playere media ca subtitrări. |
| Markdown | `.md` | Documente text simplu, ușor de citit. |
| Word Document | `.docx` | Partajare cu utilizatorii de Microsoft Word. |
| SQLite Database | `.db` | Export complet al bazei de date pentru interogări personalizate. |

## Numele Vorbitorilor în Exporturi

Dacă ați atribuit nume afișate vorbitorilor, aceste nume sunt utilizate în toate formatele de export. Pentru a actualiza numele înainte de export, faceți mai întâi clic pe `Edit Speaker Names`. Consultați [Editarea Numelor Vorbitorilor](editing_speaker_names.md).
