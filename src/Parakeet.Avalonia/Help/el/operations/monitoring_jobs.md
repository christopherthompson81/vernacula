# Παρακολούθηση Εργασιών

Η προβολή **Πρόοδος** σάς προσφέρει μια ζωντανή εικόνα μιας εργασίας μεταγραφής που βρίσκεται σε εξέλιξη.

## Άνοιγμα της Προβολής Προόδου

- Όταν ξεκινάτε μια νέα μεταγραφή, η εφαρμογή μεταβαίνει αυτόματα στην προβολή Προόδου.
- Για μια εργασία που εκτελείται ήδη ή βρίσκεται σε ουρά, εντοπίστε την στον πίνακα **Ιστορικό Μεταγραφών** και κάντε κλικ στο `Monitor` στη στήλη **Ενέργειες**.

## Ανάγνωση της Προβολής Προόδου

| Στοιχείο | Περιγραφή |
|---|---|
| Γραμμή προόδου | Συνολικό ποσοστό ολοκλήρωσης. Εμφανίζεται αόριστη (κινούμενη) ενώ η εργασία εκκινεί ή συνεχίζεται μετά από παύση. |
| Ετικέτα ποσοστού | Αριθμητικό ποσοστό που εμφανίζεται στα δεξιά της γραμμής. |
| Μήνυμα κατάστασης | Τρέχουσα δραστηριότητα — για παράδειγμα `Audio Analysis` ή `Speech Recognition`. Εμφανίζει `Waiting in queue…` εάν η εργασία δεν έχει ξεκινήσει ακόμα. |
| Πίνακας τμημάτων | Ζωντανή ροή μεταγεγραμμένων τμημάτων με στήλες **Ομιλητής**, **Έναρξη**, **Λήξη** και **Περιεχόμενο**. Κυλά αυτόματα καθώς φτάνουν νέα τμήματα. |

## Φάσεις Προόδου

Οι φάσεις που εμφανίζονται εξαρτώνται από τη **Λειτουργία Τμηματοποίησης** που έχει επιλεγεί στις Ρυθμίσεις.

**Λειτουργία Διαχωρισμού Ομιλητών** (προεπιλογή):

1. **Ανάλυση Ήχου** — Η διαδικασία διαχωρισμού SortFormer εκτελείται σε ολόκληρο το αρχείο για τον εντοπισμό ορίων ομιλητών. Η γραμμή μπορεί να παραμένει κοντά στο 0% μέχρι να ολοκληρωθεί αυτή η φάση.
2. **Αναγνώριση Ομιλίας** — κάθε τμήμα ομιλητή μεταγράφεται. Το ποσοστό ανεβαίνει σταθερά κατά τη διάρκεια αυτής της φάσης.

**Λειτουργία Ανίχνευσης Φωνητικής Δραστηριότητας**:

1. **Ανίχνευση τμημάτων ομιλίας** — Το Silero VAD σαρώνει το αρχείο για να εντοπίσει περιοχές ομιλίας. Αυτή η φάση είναι γρήγορη.
2. **Αναγνώριση Ομιλίας** — κάθε ανιχνευμένη περιοχή ομιλίας μεταγράφεται.

Και στις δύο λειτουργίες, ο ζωντανός πίνακας τμημάτων συμπληρώνεται καθώς προχωρά η μεταγραφή.

## Πλοήγηση Μακριά

Κάντε κλικ στο `← Back to Home` για να επιστρέψετε στην αρχική οθόνη χωρίς να διακόψετε την εργασία. Η εργασία συνεχίζει να εκτελείται στο παρασκήνιο και η κατάστασή της ενημερώνεται στον πίνακα **Ιστορικό Μεταγραφών**. Κάντε ξανά κλικ στο `Monitor` οποιαδήποτε στιγμή για να επιστρέψετε στην προβολή Προόδου.
