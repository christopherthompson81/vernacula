# Ροή Εργασίας Νέας Απομαγνητοφώνησης

Χρησιμοποιήστε αυτή τη ροή εργασίας για να απομαγνητοφωνήσετε ένα μεμονωμένο ηχητικό αρχείο.

## Προαπαιτούμενα

- Όλα τα αρχεία μοντέλων πρέπει να έχουν ληφθεί. Η κάρτα **Κατάσταση Μοντέλου** πρέπει να εμφανίζει `All N model file(s) present ✓`. Δείτε [Λήψη Μοντέλων](../first_steps/downloading_models.md).

## Υποστηριζόμενες Μορφές

### Ήχος

`.wav`, `.mp3`, `.flac`, `.m4a`, `.ogg`, `.aac`, `.wma`, `.aiff`, `.aif`, `.webm`

### Βίντεο

`.mp4`, `.mov`, `.mkv`, `.avi`, `.wmv`, `.flv`, `.ts`, `.mts`, `.m2ts`, `.3gp`

Τα αρχεία βίντεο αποκωδικοποιούνται μέσω FFmpeg. Εάν ένα αρχείο βίντεο περιέχει **πολλαπλές ροές ήχου** (π.χ. πολλαπλές γλώσσες ή κομμάτια σχολιασμού), δημιουργείται αυτόματα μία εργασία απομαγνητοφώνησης για κάθε ροή.

## Βήματα

### 1. Ανοίξτε τη φόρμα Νέας Απομαγνητοφώνησης

Κάντε κλικ στο `New Transcription` στην αρχική οθόνη ή μεταβείτε στο `File > New Transcription`.

### 2. Επιλέξτε ένα αρχείο πολυμέσων

Κάντε κλικ στο `Browse…` δίπλα στο πεδίο **Αρχείο Ήχου**. Ανοίγει ένας επιλογέας αρχείων φιλτραρισμένος στις υποστηριζόμενες μορφές ήχου και βίντεο. Επιλέξτε το αρχείο σας και κάντε κλικ στο **Άνοιγμα**. Η διαδρομή του αρχείου εμφανίζεται στο πεδίο.

### 3. Ονομάστε την εργασία

Το πεδίο **Όνομα Εργασίας** συμπληρώνεται εκ των προτέρων από το όνομα του αρχείου. Επεξεργαστείτε το εάν θέλετε διαφορετική ετικέτα — αυτό το όνομα εμφανίζεται στο Ιστορικό Απομαγνητοφωνήσεων στην αρχική οθόνη.

### 4. Ξεκινήστε την απομαγνητοφώνηση

Κάντε κλικ στο `Start Transcription`. Η εφαρμογή μεταβαίνει στην προβολή **Προόδου**.

Για να επιστρέψετε χωρίς να ξεκινήσετε, κάντε κλικ στο `← Back`.

## Τι Συμβαίνει Στη Συνέχεια

Η εργασία εκτελείται σε δύο φάσεις που εμφανίζονται στη γραμμή προόδου:

1. **Ανάλυση Ήχου** — διαχωρισμός ομιλητών: αναγνώριση του ποιος μιλά και πότε.
2. **Αναγνώριση Ομιλίας** — μετατροπή της ομιλίας σε κείμενο τμήμα προς τμήμα.

Τα απομαγνητοφωνημένα τμήματα εμφανίζονται στον πίνακα σε πραγματικό χρόνο καθώς παράγονται. Όταν ολοκληρωθεί η επεξεργασία, η εφαρμογή μεταβαίνει αυτόματα στην προβολή **Αποτελεσμάτων**.

Εάν προσθέσετε μια εργασία ενώ μια άλλη εκτελείται ήδη, η νέα εργασία θα εμφανίσει κατάσταση `queued` και θα ξεκινήσει όταν ολοκληρωθεί η τρέχουσα εργασία. Δείτε [Παρακολούθηση Εργασιών](monitoring_jobs.md).
