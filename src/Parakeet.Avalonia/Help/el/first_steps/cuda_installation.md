# Εγκατάσταση CUDA και cuDNN για Επιτάχυνση GPU

Το Parakeet Transcription μπορεί να χρησιμοποιεί μια NVIDIA GPU για να επιταχύνει σημαντικά τη μεταγραφή. Η επιτάχυνση GPU απαιτεί την εγκατάσταση του NVIDIA CUDA Toolkit και των βιβλιοθηκών χρόνου εκτέλεσης cuDNN στο σύστημά σας.

## Απαιτήσεις

- Μια NVIDIA GPU με υποστήριξη CUDA (συνιστάται GeForce GTX σειρά 10 ή νεότερη).
- Windows 10 ή 11 (64-bit).
- Τα αρχεία μοντέλου πρέπει να έχουν ήδη ληφθεί. Δείτε [Λήψη Μοντέλων](downloading_models.md).

## Βήματα Εγκατάστασης

### 1. Εγκατάσταση του CUDA Toolkit

Κατεβάστε και εκτελέστε το πρόγραμμα εγκατάστασης του CUDA Toolkit από τον ιστότοπο NVIDIA για προγραμματιστές. Κατά την εγκατάσταση, αποδεχτείτε τις προεπιλεγμένες διαδρομές. Το πρόγραμμα εγκατάστασης ορίζει αυτόματα τη μεταβλητή περιβάλλοντος `CUDA_PATH` — το Parakeet χρησιμοποιεί αυτή τη μεταβλητή για να εντοπίσει τις βιβλιοθήκες CUDA.

### 2. Εγκατάσταση του cuDNN

Κατεβάστε το αρχείο ZIP του cuDNN για την εγκατεστημένη έκδοση CUDA από τον ιστότοπο NVIDIA για προγραμματιστές. Αποσυμπιέστε το αρχείο και αντιγράψτε τα περιεχόμενα των φακέλων `bin`, `include` και `lib` στους αντίστοιχους φακέλους του καταλόγου εγκατάστασης του CUDA Toolkit (η διαδρομή που εμφανίζεται από τη μεταβλητή `CUDA_PATH`).

Εναλλακτικά, εγκαταστήστε το cuDNN χρησιμοποιώντας το πρόγραμμα εγκατάστασης NVIDIA cuDNN, εάν είναι διαθέσιμο για την έκδοση CUDA που διαθέτετε.

### 3. Επανεκκίνηση της Εφαρμογής

Κλείστε και ανοίξτε ξανά το Parakeet Transcription μετά την εγκατάσταση. Η εφαρμογή ελέγχει για CUDA κατά την εκκίνηση.

## Κατάσταση GPU στις Ρυθμίσεις

Ανοίξτε `Settings…` από τη γραμμή μενού και εντοπίστε την ενότητα **Hardware & Performance**. Κάθε στοιχείο εμφανίζει ένα σύμβολο επιλογής (✓) όταν εντοπιστεί:

| Στοιχείο | Τι σημαίνει |
|---|---|
| Όνομα GPU και VRAM | Η NVIDIA GPU σας εντοπίστηκε |
| CUDA Toolkit ✓ | Οι βιβλιοθήκες CUDA εντοπίστηκαν μέσω `CUDA_PATH` |
| cuDNN ✓ | Τα αρχεία DLL χρόνου εκτέλεσης cuDNN εντοπίστηκαν |
| CUDA Acceleration ✓ | Το ONNX Runtime φόρτωσε τον πάροχο εκτέλεσης CUDA |

Εάν κάποιο στοιχείο λείπει μετά την εγκατάσταση, κάντε κλικ στο `Re-check` για να εκτελεστεί εκ νέου η ανίχνευση υλικού χωρίς επανεκκίνηση της εφαρμογής.

Το παράθυρο Ρυθμίσεων παρέχει επίσης απευθείας συνδέσμους λήψης για το CUDA Toolkit και το cuDNN, εάν δεν έχουν εγκατασταθεί ακόμα.

### Αντιμετώπιση Προβλημάτων

Εάν το `CUDA Acceleration` δεν εμφανίζει σύμβολο επιλογής, βεβαιωθείτε ότι:

- Η μεταβλητή περιβάλλοντος `CUDA_PATH` έχει οριστεί (ελέγξτε `System > Advanced system settings > Environment Variables`).
- Τα αρχεία DLL του cuDNN βρίσκονται σε κατάλογο που περιλαμβάνεται στο `PATH` του συστήματός σας ή μέσα στον φάκελο `bin` του CUDA.
- Το πρόγραμμα οδήγησης της GPU σας είναι ενημερωμένο.

### Μέγεθος Παρτίδας

Όταν το CUDA είναι ενεργό, η ενότητα **Hardware & Performance** εμφανίζει επίσης το τρέχον δυναμικό ανώτατο όριο παρτίδας — τον μέγιστο αριθμό δευτερολέπτων ήχου που επεξεργάζεται σε μία εκτέλεση GPU. Αυτή η τιμή υπολογίζεται από τη διαθέσιμη VRAM μετά τη φόρτωση των μοντέλων και προσαρμόζεται αυτόματα εάν αλλάξει η διαθέσιμη μνήμη.

## Εκτέλεση Χωρίς GPU

Εάν το CUDA δεν είναι διαθέσιμο, το Parakeet μεταπίπτει αυτόματα στην επεξεργασία μέσω CPU. Η μεταγραφή εξακολουθεί να λειτουργεί, αλλά θα είναι πιο αργή, ιδιαίτερα για μεγάλα αρχεία ήχου.
