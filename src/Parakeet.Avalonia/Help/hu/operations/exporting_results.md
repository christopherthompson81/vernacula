# Eredmények vagy átiratok exportálása

A kész átiratot több fájlformátumba is exportálhatja, hogy más alkalmazásokban is felhasználhassa.

## Az exportálás menete

1. Nyisson meg egy kész feladatot. Lásd: [Kész feladatok betöltése](loading_completed_jobs.md).
2. Az **Eredmények** nézetben kattintson az `Export Transcript` gombra.
3. Megnyílik az **Export Transcript** párbeszédablak. Válasszon formátumot a **Format** legördülő listából.
4. Kattintson a `Save` gombra. Megnyílik a fájlmentési párbeszédablak.
5. Válasszon célmappát és fájlnevet, majd kattintson a **Save** gombra.

A párbeszédablak alján egy megerősítő üzenet jelenik meg, amely mutatja a mentett fájl teljes elérési útját.

## Elérhető formátumok

| Formátum | Kiterjesztés | Leginkább alkalmas |
|---|---|---|
| Excel | `.xlsx` | Táblázatos elemzés előadói, időbélyeg- és tartalomoszlopokkal. |
| CSV | `.csv` | Importálás bármely táblázatkezelőbe vagy adatkezelő eszközbe. |
| JSON | `.json` | Programozott feldolgozás. |
| SRT Subtitles | `.srt` | Videószerkesztőkbe vagy médialejátszókba töltés feliratként. |
| Markdown | `.md` | Olvasható egyszerű szöveges dokumentumok. |
| Word Document | `.docx` | Megosztás Microsoft Word felhasználókkal. |
| SQLite Database | `.db` | Teljes adatbázis-exportálás egyéni lekérdezésekhez. |

## Előadók nevei az exportált fájlokban

Ha megjelenítési neveket rendelt az előadókhoz, ezeket a neveket az összes exportformátum használja. Ha az exportálás előtt szeretné frissíteni a neveket, először kattintson az `Edit Speaker Names` gombra. Lásd: [Előadók neveinek szerkesztése](editing_speaker_names.md).
