# Tulemuste või ärakirjade eksportimine

Saate lõpetatud ärakirja eksportida mitmesse failivormingusse, et kasutada seda teistes rakendustes.

## Kuidas eksportida

1. Avage lõpetatud töö. Vt [Lõpetatud tööde laadimine](loading_completed_jobs.md).
2. Klõpsake vaates **Tulemused** nupul `Export Transcript`.
3. Avaneb dialoog **Export Transcript**. Valige vorming rippmenüüst **Format**.
4. Klõpsake `Save`. Avaneb faili salvestamise dialoog.
5. Valige sihtkaust ja failinimi, seejärel klõpsake **Save**.

Dialoogi allosas kuvatakse kinnitussõnum koos salvestatud faili täieliku teega.

## Saadaolevad vormingud

| Vorming | Laiend | Sobib kõige paremini |
|---|---|---|
| Excel | `.xlsx` | Tabelarvutuse analüüs veergudega kõneleja, ajatemplite ja sisu jaoks. |
| CSV | `.csv` | Import mis tahes tabelarvutus- või andmetööriista. |
| JSON | `.json` | Programmatiline töötlemine. |
| SRT Subtitles | `.srt` | Laadimine videotöötlusprogrammidesse või meediapleieritesse subtiitritena. |
| Markdown | `.md` | Loetavad lihttekstdokumendid. |
| Word Document | `.docx` | Jagamine Microsoft Wordi kasutajatega. |
| SQLite Database | `.db` | Täielik andmebaasi eksport kohandatud päringute jaoks. |

## Kõnelejate nimed ekspordis

Kui olete kõnelejatele kuvatavad nimed määranud, kasutatakse neid nimesid kõigis ekspordivormingutes. Enne eksportimist nimede uuendamiseks klõpsake esmalt `Edit Speaker Names`. Vt [Kõnelejate nimede muutmine](editing_speaker_names.md).
