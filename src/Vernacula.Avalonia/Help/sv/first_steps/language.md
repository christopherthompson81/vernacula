# Välja språk

Vernacula-Desktop stöder flera visningsspråk för sitt användargränssnitt.

## Hur du ändrar språket

Öppna `Settings…` från menyraden. I avsnittet **Language** klickar du på det språk du vill använda. Gränssnittet uppdateras omedelbart — ingen omstart krävs.

## Automatisk språkidentifiering

Vid första start identifierar applikationen operativsystemets språk. Om en matchande språkinställning finns tillgänglig väljs den automatiskt. Du kan när som helst ändra detta via menyn.

## Obs angående transkriberingsspråk

Visningsspråket styr endast applikationens gränssnitt. Transkriptionmodellen känner igen tal oberoende av denna inställning — den är inte begränsad till det valda visningsspråket.
