# Choisir une langue

Parakeet Transcription prend en charge plusieurs langues d'affichage pour son interface utilisateur.

## Comment modifier la langue

Ouvrez `Settings…` depuis la barre de menus. Dans la section **Language**, cliquez sur la langue souhaitée. L'interface se met à jour immédiatement — aucun redémarrage n'est nécessaire.

## Détection automatique de la langue

Au premier lancement, l'application détecte la langue de votre système d'exploitation. Si un paramètre régional correspondant est disponible, il est sélectionné automatiquement. Vous pouvez modifier ce choix à tout moment depuis le menu.

## Remarque sur la langue de transcription

Le paramètre de langue d'affichage contrôle uniquement l'interface de l'application. Le modèle de transcription reconnaît la parole indépendamment de ce paramètre — il n'est pas limité à la langue d'affichage sélectionnée.
