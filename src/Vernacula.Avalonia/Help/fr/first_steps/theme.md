# Choisir un thème

Parakeet Transcription propose des thèmes Sombre et Clair. Le thème actif est indiqué par une coche dans le menu.

## Comment changer de thème

Ouvrez `Settings…` depuis la barre de menus. Dans la section **Appearance**, sélectionnez `Dark` ou `Light`. Le thème change immédiatement — aucun redémarrage n'est nécessaire.
