# Picking a Language

Parakeet Transcription supports multiple display languages for its user interface.

## How to Change the Language

Open `Settings…` from the menu bar. In the **Language** section, click the language you want. The interface updates immediately — no restart is required.

## Automatic Language Detection

On first launch the application detects your operating system's language. If a matching locale is available it is selected automatically. You can override this at any time using the menu.

## Note on Transcription Language

The display language setting controls only the application interface. The transcription model itself recognises speech regardless of this setting — it is not limited to the selected display language.
