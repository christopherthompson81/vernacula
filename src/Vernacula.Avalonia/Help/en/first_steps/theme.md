# Picking a Theme

Vernacula-Desktop offers Dark and Light themes. The active theme is indicated by a checkmark in the menu.

## How to Change the Theme

Open `Settings…` from the menu bar. In the **Appearance** section, select either `Dark` or `Light`. The theme changes immediately — no restart is required.
