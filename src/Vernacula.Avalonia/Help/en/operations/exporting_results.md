# Exporting Results or Transcripts

You can export a completed transcript to several file formats for use in other applications.

## How to Export

1. Open a completed job. See [Loading Completed Jobs](loading_completed_jobs.md).
2. In the **Results** view, click `Export Transcript`.
3. The **Export Transcript** dialog opens. Choose a format from the **Format** drop-down.
4. Click `Save`. A save-file dialog opens.
5. Choose a destination folder and file name, then click **Save**.

A confirmation message appears at the bottom of the dialog showing the full path of the saved file.

## Available Formats

| Format | Extension | Best for |
|---|---|---|
| Excel | `.xlsx` | Spreadsheet analysis with columns for speaker, timestamps, and content. |
| CSV | `.csv` | Import into any spreadsheet or data tool. |
| JSON | `.json` | Programmatic processing. |
| SRT Subtitles | `.srt` | Loading into video editors or media players as subtitles. |
| Markdown | `.md` | Readable plain-text documents. |
| Word Document | `.docx` | Sharing with users of Microsoft Word. |
| SQLite Database | `.db` | Full database export for custom queries. |

## Speaker Names in Exports

If you have assigned display names to speakers, those names are used in all export formats. To update names before exporting, click `Edit Speaker Names` first. See [Editing Speaker Names](editing_speaker_names.md).
