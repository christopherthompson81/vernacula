# Selectarea unei limbi

Vernacula-Desktop acceptă mai multe limbi de afișare pentru interfața sa.

## Cum se schimbă limba

Deschideți `Settings…` din bara de meniu. În secțiunea **Language**, faceți clic pe limba dorită. Interfața se actualizează imediat — nu este necesară repornirea aplicației.

## Detectarea automată a limbii

La prima lansare, aplicația detectează limba sistemului de operare. Dacă există o localizare corespunzătoare, aceasta este selectată automat. Puteți modifica oricând această setare din meniu.

## Notă privind limba de transcriere

Setarea limbii de afișare controlează doar interfața aplicației. Modelul de transcriere în sine recunoaște vorbirea indiferent de această setare — nu este limitat la limba de afișare selectată.
