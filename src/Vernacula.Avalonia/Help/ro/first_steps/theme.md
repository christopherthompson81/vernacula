# Alegerea unei teme

Vernacula-Desktop oferă temele Întunecat și Luminos. Tema activă este indicată printr-o bifă în meniu.

## Cum să schimbați tema

Deschideți `Settings…` din bara de meniu. În secțiunea **Appearance**, selectați `Dark` sau `Light`. Tema se schimbă imediat — nu este necesară repornirea aplicației.
