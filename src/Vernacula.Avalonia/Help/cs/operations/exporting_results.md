# Export výsledků nebo přepisů

Dokončený přepis můžete exportovat do několika formátů souborů pro použití v jiných aplikacích.

## Jak exportovat

1. Otevřete dokončenou úlohu. Viz [Načítání dokončených úloh](loading_completed_jobs.md).
2. V zobrazení **Výsledky** klikněte na `Export Transcript`.
3. Otevře se dialog **Export Transcript**. V rozevíracím seznamu **Format** vyberte požadovaný formát.
4. Klikněte na `Save`. Otevře se dialog pro uložení souboru.
5. Vyberte cílovou složku a název souboru, poté klikněte na **Uložit**.

V dolní části dialogu se zobrazí potvrzovací zpráva s úplnou cestou k uloženému souboru.

## Dostupné formáty

| Formát | Přípona | Nejvhodnější pro |
|---|---|---|
| Excel | `.xlsx` | Analýzu v tabulkovém procesoru se sloupci pro mluvčího, časové značky a obsah. |
| CSV | `.csv` | Import do libovolného tabulkového procesoru nebo nástroje pro práci s daty. |
| JSON | `.json` | Programové zpracování. |
| SRT titulky | `.srt` | Načtení do video editorů nebo přehrávačů médií jako titulky. |
| Markdown | `.md` | Čitelné dokumenty ve formátu prostého textu. |
| Dokument Word | `.docx` | Sdílení s uživateli aplikace Microsoft Word. |
| Databáze SQLite | `.db` | Úplný export databáze pro vlastní dotazy. |

## Jména mluvčích v exportech

Pokud jste mluvčím přiřadili zobrazovaná jména, tato jména budou použita ve všech formátech exportu. Chcete-li jména před exportem upravit, nejprve klikněte na `Edit Speaker Names`. Viz [Úprava jmen mluvčích](editing_speaker_names.md).
