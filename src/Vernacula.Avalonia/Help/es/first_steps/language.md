# Selección de idioma

Parakeet Transcription admite varios idiomas de visualización para su interfaz de usuario.

## Cómo cambiar el idioma

Abre `Settings…` desde la barra de menú. En la sección **Language**, haz clic en el idioma que desees. La interfaz se actualiza de inmediato: no es necesario reiniciar la aplicación.

## Detección automática del idioma

En el primer inicio, la aplicación detecta el idioma del sistema operativo. Si hay una configuración regional coincidente disponible, se selecciona automáticamente. Puedes cambiarla en cualquier momento desde el menú.

## Nota sobre el idioma de transcripción

La configuración del idioma de visualización controla únicamente la interfaz de la aplicación. El modelo de transcripción en sí reconoce el habla independientemente de esta configuración; no está limitado al idioma de visualización seleccionado.
