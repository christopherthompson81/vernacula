# Экспорт результатов или транскриптов

Вы можете экспортировать готовый транскрипт в несколько форматов файлов для использования в других приложениях.

## Как выполнить экспорт

1. Откройте завершённое задание. См. [Загрузка завершённых заданий](loading_completed_jobs.md).
2. В представлении **Результаты** нажмите `Export Transcript`.
3. Откроется диалоговое окно **Export Transcript**. Выберите формат из раскрывающегося списка **Format**.
4. Нажмите `Save`. Откроется диалоговое окно сохранения файла.
5. Выберите папку назначения и имя файла, затем нажмите **Save**.

В нижней части диалогового окна появится сообщение с подтверждением, содержащее полный путь к сохранённому файлу.

## Доступные форматы

| Формат | Расширение | Лучше всего подходит для |
|---|---|---|
| Excel | `.xlsx` | Анализ данных в таблицах со столбцами для имён говорящих, временных меток и содержимого. |
| CSV | `.csv` | Импорт в любую таблицу или инструмент обработки данных. |
| JSON | `.json` | Программная обработка данных. |
| SRT Subtitles | `.srt` | Загрузка в видеоредакторы или медиаплееры в качестве субтитров. |
| Markdown | `.md` | Читаемые текстовые документы. |
| Word Document | `.docx` | Совместное использование с пользователями Microsoft Word. |
| SQLite Database | `.db` | Полный экспорт базы данных для пользовательских запросов. |

## Имена говорящих в экспортируемых файлах

Если вы присвоили говорящим отображаемые имена, эти имена используются во всех форматах экспорта. Чтобы обновить имена перед экспортом, сначала нажмите `Edit Speaker Names`. См. [Редактирование имён говорящих](editing_speaker_names.md).
