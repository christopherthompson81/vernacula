# Настройки

Окно **Настройки** позволяет управлять конфигурацией оборудования, моделями, режимом сегментации, поведением редактора, внешним видом и языком интерфейса. Откройте его через строку меню: `Settings…`.

## Оборудование и производительность

В этом разделе отображается состояние вашего GPU NVIDIA и программного стека CUDA, а также сообщается о динамическом лимите пакетной обработки, используемом при транскрипции на GPU.

| Элемент | Описание |
|---|---|
| Название GPU и объём VRAM | Обнаруженный GPU NVIDIA и доступная видеопамять. |
| CUDA Toolkit | Наличие библиотек среды выполнения CUDA, найденных через `CUDA_PATH`. |
| cuDNN | Наличие доступных DLL-библиотек среды выполнения cuDNN. |
| Ускорение CUDA | Успешность загрузки провайдера выполнения CUDA в ONNX Runtime. |

Нажмите `Re-check`, чтобы повторно выполнить определение оборудования без перезапуска приложения — это удобно после установки CUDA или cuDNN.

Прямые ссылки для загрузки CUDA Toolkit и cuDNN отображаются в том случае, если эти компоненты не обнаружены.

Сообщение о **лимите пакетной обработки** показывает, сколько секунд аудио обрабатывается за один запуск на GPU. Это значение вычисляется на основе свободного объёма VRAM после загрузки моделей и корректируется автоматически.

Полные инструкции по настройке CUDA см. в разделе [Установка CUDA и cuDNN](cuda_installation.md).

## Модели

В этом разделе осуществляется управление файлами моделей ИИ, необходимыми для транскрипции.

- **Загрузить недостающие модели** — загружает файлы моделей, которые ещё отсутствуют на диске. Ход загрузки каждого файла отслеживается с помощью индикатора выполнения и строки состояния.
- **Проверить обновления** — проверяет наличие более новых весов моделей. При обнаружении обновлённых весов на главном экране также автоматически появляется баннер с уведомлением.

## Режим сегментации

Управляет тем, как аудио разбивается на сегменты перед распознаванием речи.

| Режим | Описание |
|---|---|
| **Диаризация дикторов** | Использует модель SortFormer для определения отдельных дикторов и присвоения меток каждому сегменту. Лучше всего подходит для интервью, совещаний и записей с несколькими участниками. |
| **Определение голосовой активности** | Использует Silero VAD для обнаружения только речевых фрагментов — без меток дикторов. Работает быстрее диаризации и хорошо подходит для аудио с одним диктором. |

## Редактор транскриптов

**Режим воспроизведения по умолчанию** — задаёт режим воспроизведения, который используется при открытии редактора транскриптов. Изменить его можно также непосредственно в редакторе в любой момент. Описание каждого режима см. в разделе [Редактирование транскриптов](../operations/editing_transcripts.md).

## Внешний вид

Выберите тему **Тёмная** или **Светлая**. Изменение применяется немедленно. См. [Выбор темы](theme.md).

## Язык

Выберите язык отображения интерфейса приложения. Изменение применяется немедленно. См. [Выбор языка](language.md).
