# Выбор языка

Vernacula-Desktop поддерживает несколько языков отображения пользовательского интерфейса.

## Как изменить язык

Откройте `Settings…` в строке меню. В разделе **Language** нажмите на нужный язык. Интерфейс обновится немедленно — перезапуск не требуется.

## Автоматическое определение языка

При первом запуске приложение определяет язык вашей операционной системы. Если соответствующий языковой стандарт доступен, он выбирается автоматически. Вы можете изменить его в любое время с помощью меню.

## Примечание о языке транскрипции

Настройка языка отображения влияет только на интерфейс приложения. Сама модель транскрипции распознаёт речь независимо от этой настройки — она не ограничена выбранным языком отображения.
