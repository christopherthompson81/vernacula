# Додавання кількох аудіофайлів до черги

Використовуйте **Масове додавання завдань**, щоб поставити кілька аудіо- або відеофайлів у чергу для транскрипції за один крок. Програма обробляє їх по одному в порядку додавання.

## Передумови

- Усі файли моделі мають бути завантажені. На картці **Стан моделі** має відображатися `All N model file(s) present ✓`. Див. [Завантаження моделей](../first_steps/downloading_models.md).

## Як масово додати завдання

1. На головному екрані натисніть `Bulk Add Jobs`.
2. Відкриється вікно вибору файлів. Виберіть один або кілька аудіо- чи відеофайлів — утримуйте `Ctrl` або `Shift` для вибору кількох файлів.
3. Натисніть **Відкрити**. Кожен вибраний файл буде додано до таблиці **Історія транскрипцій** як окреме завдання.

> **Відеофайли з кількома аудіопотоками:** якщо відеофайл містить більше одного аудіопотоку (наприклад, кілька мов або коментар режисера), програма автоматично створює одне завдання для кожного потоку.

## Назви завдань

Кожне завдання автоматично отримує назву відповідно до імені аудіофайлу. Ви можете перейменувати завдання будь-коли, клацнувши його назву в стовпці **Title** таблиці історії транскрипцій, відредагувавши текст і натиснувши `Enter` або клацнувши в інше місце.

## Поведінка черги

- Якщо жодне завдання наразі не виконується, перший файл розпочне обробку негайно, а решта відображатимуться зі статусом `queued`.
- Якщо завдання вже виконується, усі щойно додані файли відображатимуться зі статусом `queued` і запускатимуться автоматично по черзі.
- Щоб стежити за активним завданням, натисніть `Monitor` у стовпці **Actions** для цього завдання. Див. [Моніторинг завдань](monitoring_jobs.md).
- Щоб призупинити або видалити завдання з черги до його початку, використовуйте кнопки `Pause` або `Remove` у стовпці **Actions** для цього завдання. Див. [Призупинення, відновлення або видалення завдань](pausing_resuming_removing.md).
