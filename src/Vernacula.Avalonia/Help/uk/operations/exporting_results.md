# Експорт результатів або транскриптів

Ви можете експортувати завершений транскрипт у кілька форматів файлів для використання в інших програмах.

## Як виконати експорт

1. Відкрийте завершене завдання. Див. [Завантаження завершених завдань](loading_completed_jobs.md).
2. У поданні **Результати** натисніть `Export Transcript`.
3. Відкриється діалогове вікно **Export Transcript**. Виберіть формат зі спадного списку **Format**.
4. Натисніть `Save`. Відкриється діалогове вікно збереження файлу.
5. Виберіть папку призначення та ім'я файлу, а потім натисніть **Save**.

Внизу діалогового вікна з'явиться повідомлення про підтвердження з повним шляхом до збереженого файлу.

## Доступні формати

| Формат | Розширення | Найкраще підходить для |
|---|---|---|
| Excel | `.xlsx` | Аналіз у електронних таблицях зі стовпцями для мовця, міток часу та вмісту. |
| CSV | `.csv` | Імпорт у будь-яку електронну таблицю або інструмент роботи з даними. |
| JSON | `.json` | Програмна обробка. |
| SRT Subtitles | `.srt` | Завантаження у відеоредактори або медіаплеєри як субтитри. |
| Markdown | `.md` | Читабельні текстові документи. |
| Word Document | `.docx` | Обмін з користувачами Microsoft Word. |
| SQLite Database | `.db` | Повний експорт бази даних для власних запитів. |

## Імена мовців в експортованих файлах

Якщо ви призначили відображувані імена мовцям, ці імена використовуються в усіх форматах експорту. Щоб оновити імена перед експортом, спочатку натисніть `Edit Speaker Names`. Див. [Редагування імен мовців](editing_speaker_names.md).
