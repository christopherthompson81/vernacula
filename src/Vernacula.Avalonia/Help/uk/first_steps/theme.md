# Вибір теми

Parakeet Transcription пропонує темну та світлу теми. Активна тема позначена галочкою в меню.

## Як змінити тему

Відкрийте `Settings…` у рядку меню. У розділі **Appearance** виберіть `Dark` або `Light`. Тема змінюється одразу — перезапуск не потрібен.
