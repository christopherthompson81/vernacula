# Вибір точності ваг моделі

Точність моделі визначає числовий формат, який використовується для ваг моделі ШІ. Вона впливає на розмір завантаження, використання пам'яті та точність розпізнавання.

## Варіанти точності

### INT8 (менший розмір завантаження)

- Менші файли моделі — швидше завантаження та менше місця на диску.
- Дещо нижча точність для деяких аудіозаписів.
- Рекомендовано, якщо у вас обмежено місце на диску або повільне інтернет-з'єднання.

### FP32 (вища точність)

- Більші файли моделі.
- Вища точність, особливо для складних аудіозаписів із акцентами або фоновим шумом.
- Рекомендовано, коли точність є пріоритетом і у вас достатньо місця на диску.
- Обов'язковий для прискорення на CUDA GPU — GPU завжди використовує FP32 незалежно від цього налаштування.

## Як змінити точність

Відкрийте `Settings…` у рядку меню, перейдіть до розділу **Models** і виберіть `INT8 (smaller download)` або `FP32 (more accurate)`.

## Після зміни точності

Зміна точності потребує іншого набору файлів моделі. Якщо файли моделі для нової точності ще не завантажено, натисніть `Download Missing Models` у налаштуваннях. Раніше завантажені файли для іншої точності зберігаються на диску, і їх не потрібно завантажувати повторно, якщо ви повернетесь до попередньої точності.
