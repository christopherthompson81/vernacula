# Wybór języka

Vernacula-Desktop obsługuje wiele języków wyświetlania interfejsu użytkownika.

## Jak zmienić język

Otwórz `Settings…` z paska menu. W sekcji **Language** kliknij wybrany język. Interfejs zostaje zaktualizowany natychmiast — ponowne uruchomienie nie jest wymagane.

## Automatyczne wykrywanie języka

Przy pierwszym uruchomieniu aplikacja wykrywa język systemu operacyjnego. Jeśli dostępne jest odpowiednie ustawienie regionalne, zostaje ono wybrane automatycznie. Możesz je zmienić w dowolnym momencie za pomocą menu.

## Uwaga dotycząca języka transkrypcji

Ustawienie języka wyświetlania kontroluje wyłącznie interfejs aplikacji. Sam model transkrypcji rozpoznaje mowę niezależnie od tego ustawienia — nie jest on ograniczony do wybranego języka wyświetlania.
