# Exportando Resultados ou Transcrições

Você pode exportar uma transcrição concluída para vários formatos de arquivo para uso em outros aplicativos.

## Como Exportar

1. Abra um trabalho concluído. Consulte [Carregando Trabalhos Concluídos](loading_completed_jobs.md).
2. Na visualização **Resultados**, clique em `Export Transcript`.
3. A caixa de diálogo **Export Transcript** é aberta. Escolha um formato no menu suspenso **Format**.
4. Clique em `Save`. Uma caixa de diálogo para salvar arquivo é aberta.
5. Escolha uma pasta de destino e um nome de arquivo e, em seguida, clique em **Save**.

Uma mensagem de confirmação aparece na parte inferior da caixa de diálogo mostrando o caminho completo do arquivo salvo.

## Formatos Disponíveis

| Formato | Extensão | Ideal para |
|---|---|---|
| Excel | `.xlsx` | Análise em planilha com colunas para locutor, marcações de tempo e conteúdo. |
| CSV | `.csv` | Importação em qualquer planilha ou ferramenta de dados. |
| JSON | `.json` | Processamento programático. |
| Legendas SRT | `.srt` | Carregamento em editores de vídeo ou reprodutores de mídia como legendas. |
| Markdown | `.md` | Documentos em texto simples e legível. |
| Documento Word | `.docx` | Compartilhamento com usuários do Microsoft Word. |
| Banco de Dados SQLite | `.db` | Exportação completa do banco de dados para consultas personalizadas. |

## Nomes de Locutores nas Exportações

Se você tiver atribuído nomes de exibição aos locutores, esses nomes serão usados em todos os formatos de exportação. Para atualizar os nomes antes de exportar, clique em `Edit Speaker Names` primeiro. Consulte [Editando Nomes de Locutores](editing_speaker_names.md).
