# Escolhendo um Tema

O Parakeet Transcription oferece os temas Escuro e Claro. O tema ativo é indicado por uma marca de seleção no menu.

## Como Alterar o Tema

Abra `Settings…` na barra de menus. Na seção **Appearance**, selecione `Dark` ou `Light`. O tema é alterado imediatamente — não é necessário reiniciar o aplicativo.
