# Escolhendo um Idioma

O Parakeet Transcription oferece suporte a vários idiomas de exibição para sua interface de usuário.

## Como Alterar o Idioma

Abra `Settings…` na barra de menus. Na seção **Language**, clique no idioma desejado. A interface é atualizada imediatamente — não é necessário reiniciar.

## Detecção Automática de Idioma

Na primeira inicialização, o aplicativo detecta o idioma do seu sistema operacional. Se uma localidade correspondente estiver disponível, ela será selecionada automaticamente. Você pode substituir essa configuração a qualquer momento pelo menu.

## Observação sobre o Idioma de Transcrição

A configuração de idioma de exibição controla apenas a interface do aplicativo. O modelo de transcrição em si reconhece a fala independentemente dessa configuração — ele não está limitado ao idioma de exibição selecionado.
