# Esportazione dei risultati o delle trascrizioni

È possibile esportare una trascrizione completata in diversi formati di file per utilizzarla in altre applicazioni.

## Come esportare

1. Apri un lavoro completato. Consulta [Caricamento dei lavori completati](loading_completed_jobs.md).
2. Nella visualizzazione **Risultati**, fai clic su `Export Transcript`.
3. Si apre la finestra di dialogo **Export Transcript**. Scegli un formato dal menu a discesa **Format**.
4. Fai clic su `Save`. Si apre una finestra di dialogo per il salvataggio del file.
5. Scegli una cartella di destinazione e un nome per il file, quindi fai clic su **Save**.

Nella parte inferiore della finestra di dialogo viene visualizzato un messaggio di conferma con il percorso completo del file salvato.

## Formati disponibili

| Formato | Estensione | Ideale per |
|---|---|---|
| Excel | `.xlsx` | Analisi in foglio di calcolo con colonne per relatore, timestamp e contenuto. |
| CSV | `.csv` | Importazione in qualsiasi foglio di calcolo o strumento per i dati. |
| JSON | `.json` | Elaborazione programmatica. |
| Sottotitoli SRT | `.srt` | Caricamento in editor video o lettori multimediali come sottotitoli. |
| Markdown | `.md` | Documenti in testo normale leggibili. |
| Documento Word | `.docx` | Condivisione con utenti di Microsoft Word. |
| Database SQLite | `.db` | Esportazione completa del database per query personalizzate. |

## Nomi dei relatori nelle esportazioni

Se hai assegnato nomi visualizzati ai relatori, tali nomi vengono utilizzati in tutti i formati di esportazione. Per aggiornare i nomi prima di esportare, fai clic su `Edit Speaker Names`. Consulta [Modifica dei nomi dei relatori](editing_speaker_names.md).
