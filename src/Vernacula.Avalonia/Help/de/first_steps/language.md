# Eine Sprache auswählen

Vernacula-Desktop unterstützt mehrere Anzeigesprachen für seine Benutzeroberfläche.

## So ändern Sie die Sprache

Öffnen Sie `Settings…` in der Menüleiste. Klicken Sie im Abschnitt **Language** auf die gewünschte Sprache. Die Benutzeroberfläche wird sofort aktualisiert – ein Neustart ist nicht erforderlich.

## Automatische Spracherkennung

Beim ersten Start erkennt die Anwendung die Sprache Ihres Betriebssystems. Ist ein passendes Gebietsschema verfügbar, wird es automatisch ausgewählt. Sie können dies jederzeit über das Menü ändern.

## Hinweis zur Transkriptionssprache

Die Anzeigesprache steuert ausschließlich die Benutzeroberfläche der Anwendung. Das Transkriptionsmodell selbst erkennt Sprache unabhängig von dieser Einstellung – es ist nicht auf die ausgewählte Anzeigesprache beschränkt.
