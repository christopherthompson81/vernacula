# Επεξεργασία Ονομάτων Ομιλητών

Η μηχανή απομαγνητοφώνησης αναθέτει αυτόματα σε κάθε ομιλητή ένα γενικό αναγνωριστικό (για παράδειγμα, `speaker_0`, `speaker_1`). Μπορείτε να τα αντικαταστήσετε με πραγματικά ονόματα, τα οποία θα εμφανίζονται σε όλο το αντίγραφο κειμένου και σε οποιοδήποτε εξαγόμενο αρχείο.

## Πώς να Επεξεργαστείτε τα Ονόματα Ομιλητών

1. Ανοίξτε μια ολοκληρωμένη εργασία. Δείτε [Φόρτωση Ολοκληρωμένων Εργασιών](loading_completed_jobs.md).
2. Στην προβολή **Results**, κάντε κλικ στο `Edit Speaker Names`.
3. Ανοίγει το παράθυρο διαλόγου **Edit Speaker Names** με δύο στήλες:
   - **Speaker ID** — η αρχική ετικέτα που ανατέθηκε από το μοντέλο (μόνο για ανάγνωση).
   - **Display Name** — το όνομα που εμφανίζεται στο αντίγραφο κειμένου (επεξεργάσιμο).
4. Κάντε κλικ σε ένα κελί της στήλης **Display Name** και πληκτρολογήστε το όνομα του ομιλητή.
5. Πατήστε `Tab` ή κάντε κλικ σε άλλη γραμμή για να μεταβείτε στον επόμενο ομιλητή.
6. Κάντε κλικ στο `Save` για να εφαρμόσετε τις αλλαγές ή στο `Cancel` για να τις απορρίψετε.

## Πού Εμφανίζονται τα Ονόματα

Τα ενημερωμένα ονόματα εμφάνισης αντικαθιστούν τα γενικά αναγνωριστικά:

- Στον πίνακα τμημάτων της προβολής Results.
- Σε όλα τα εξαγόμενα αρχεία (Excel, CSV, SRT, Markdown, Word, JSON, SQLite).

## Εκ Νέου Επεξεργασία Ονομάτων

Μπορείτε να ανοίξετε ξανά το παράθυρο διαλόγου Edit Speaker Names οποιαδήποτε στιγμή, εφόσον η εργασία είναι φορτωμένη στην προβολή Results. Οι αλλαγές αποθηκεύονται στην τοπική βάση δεδομένων και διατηρούνται μεταξύ των περιόδων λειτουργίας.
