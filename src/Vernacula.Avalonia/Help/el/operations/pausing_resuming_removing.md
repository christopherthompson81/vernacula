# Παύση, Συνέχιση ή Κατάργηση Εργασιών

## Παύση Εργασίας

Μπορείτε να κάνετε παύση σε μια εκτελούμενη ή σε αναμονή εργασία από δύο σημεία:

- **Προβολή προόδου** — κάντε κλικ στο `Pause` στην κάτω δεξιά γωνία ενώ παρακολουθείτε την ενεργή εργασία.
- **Πίνακας Ιστορικού Μεταγραφών** — κάντε κλικ στο `Pause` στη στήλη **Actions** οποιασδήποτε γραμμής με κατάσταση `running` ή `queued`.

Αφού κάνετε κλικ στο `Pause`, η γραμμή κατάστασης εμφανίζει `Pausing…` ενώ η εφαρμογή ολοκληρώνει την τρέχουσα μονάδα επεξεργασίας. Στη συνέχεια, η κατάσταση της εργασίας αλλάζει σε `cancelled` στον πίνακα ιστορικού.

> Η παύση αποθηκεύει όλα τα τμήματα που έχουν μεταγραφεί μέχρι εκείνη τη στιγμή. Μπορείτε να συνεχίσετε την εργασία αργότερα χωρίς να χαθεί αυτή η πρόοδος.

## Συνέχιση Εργασίας

Για να συνεχίσετε μια εργασία που έχει τεθεί σε παύση ή που απέτυχε:

1. Στην αρχική οθόνη, εντοπίστε την εργασία στον πίνακα **Transcription History**. Η κατάστασή της θα είναι `cancelled` ή `failed`.
2. Κάντε κλικ στο `Resume` στη στήλη **Actions**.
3. Η εφαρμογή επιστρέφει στην προβολή **Progress** και συνεχίζει από το σημείο που είχε σταματήσει η επεξεργασία.

Η γραμμή κατάστασης εμφανίζει στιγμιαία `Resuming…` ενώ η εργασία επανεκκινείται.

## Κατάργηση Εργασίας

Για να διαγράψετε οριστικά μια εργασία και τη μεταγραφή της από το ιστορικό:

1. Στον πίνακα **Transcription History**, κάντε κλικ στο `Remove` στη στήλη **Actions** της εργασίας που θέλετε να διαγράψετε.

Η εργασία αφαιρείται από τη λίστα και τα δεδομένα της διαγράφονται από την τοπική βάση δεδομένων. Αυτή η ενέργεια δεν μπορεί να αναιρεθεί. Τα εξαγόμενα αρχεία που έχουν αποθηκευτεί στον δίσκο δεν επηρεάζονται.
