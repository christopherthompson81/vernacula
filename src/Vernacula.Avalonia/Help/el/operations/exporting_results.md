# Εξαγωγή Αποτελεσμάτων ή Μεταγραφών

Μπορείτε να εξάγετε μια ολοκληρωμένη μεταγραφή σε διάφορες μορφές αρχείου για χρήση σε άλλες εφαρμογές.

## Πώς να Εξάγετε

1. Ανοίξτε μια ολοκληρωμένη εργασία. Δείτε [Φόρτωση Ολοκληρωμένων Εργασιών](loading_completed_jobs.md).
2. Στην προβολή **Αποτελέσματα**, κάντε κλικ στο `Export Transcript`.
3. Ανοίγει το παράθυρο διαλόγου **Export Transcript**. Επιλέξτε μια μορφή από το αναπτυσσόμενο μενού **Format**.
4. Κάντε κλικ στο `Save`. Ανοίγει ένα παράθυρο διαλόγου αποθήκευσης αρχείου.
5. Επιλέξτε φάκελο προορισμού και όνομα αρχείου, και στη συνέχεια κάντε κλικ στο **Save**.

Ένα μήνυμα επιβεβαίωσης εμφανίζεται στο κάτω μέρος του παραθύρου διαλόγου, όπου φαίνεται η πλήρης διαδρομή του αποθηκευμένου αρχείου.

## Διαθέσιμες Μορφές

| Μορφή | Επέκταση | Κατάλληλο για |
|---|---|---|
| Excel | `.xlsx` | Ανάλυση σε υπολογιστικό φύλλο με στήλες για ομιλητή, χρονικές σημάνσεις και περιεχόμενο. |
| CSV | `.csv` | Εισαγωγή σε οποιοδήποτε υπολογιστικό φύλλο ή εργαλείο δεδομένων. |
| JSON | `.json` | Επεξεργασία μέσω προγραμματισμού. |
| SRT Subtitles | `.srt` | Φόρτωση σε επεξεργαστές βίντεο ή προγράμματα αναπαραγωγής ως υπότιτλοι. |
| Markdown | `.md` | Αναγνώσιμα έγγραφα απλού κειμένου. |
| Word Document | `.docx` | Κοινή χρήση με χρήστες του Microsoft Word. |
| SQLite Database | `.db` | Πλήρης εξαγωγή βάσης δεδομένων για προσαρμοσμένα ερωτήματα. |

## Ονόματα Ομιλητών στις Εξαγωγές

Εάν έχετε αντιστοιχίσει εμφανιζόμενα ονόματα στους ομιλητές, αυτά τα ονόματα χρησιμοποιούνται σε όλες τις μορφές εξαγωγής. Για να ενημερώσετε τα ονόματα πριν από την εξαγωγή, κάντε πρώτα κλικ στο `Edit Speaker Names`. Δείτε [Επεξεργασία Ονομάτων Ομιλητών](editing_speaker_names.md).
