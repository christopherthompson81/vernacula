# Προσθήκη Πολλών Αρχείων Ήχου στην Ουρά

Χρησιμοποιήστε την επιλογή **Μαζική Προσθήκη Εργασιών** για να προσθέσετε πολλά αρχεία ήχου ή βίντεο στην ουρά μεταγραφής με ένα μόνο βήμα. Η εφαρμογή τα επεξεργάζεται ένα κάθε φορά, με τη σειρά που προστέθηκαν.

## Προϋποθέσεις

- Όλα τα αρχεία μοντέλου πρέπει να έχουν ληφθεί. Η κάρτα **Κατάσταση Μοντέλου** πρέπει να εμφανίζει `All N model file(s) present ✓`. Δείτε [Λήψη Μοντέλων](../first_steps/downloading_models.md).

## Πώς να Χρησιμοποιήσετε τη Μαζική Προσθήκη Εργασιών

1. Στην αρχική οθόνη, κάντε κλικ στο `Bulk Add Jobs`.
2. Ανοίγει ένα παράθυρο επιλογής αρχείων. Επιλέξτε ένα ή περισσότερα αρχεία ήχου ή βίντεο — κρατήστε πατημένο `Ctrl` ή `Shift` για να επιλέξετε πολλά αρχεία.
3. Κάντε κλικ στο **Άνοιγμα**. Κάθε επιλεγμένο αρχείο προστίθεται στον πίνακα **Ιστορικό Μεταγραφών** ως ξεχωριστή εργασία.

> **Αρχεία βίντεο με πολλές ροές ήχου:** Εάν ένα αρχείο βίντεο περιέχει περισσότερες από μία ροές ήχου (για παράδειγμα, πολλές γλώσσες ή ένα σχολιασμό σκηνοθέτη), η εφαρμογή δημιουργεί αυτόματα μία εργασία ανά ροή.

## Ονόματα Εργασιών

Κάθε εργασία ονομάζεται αυτόματα από το όνομα του αρχείου ήχου της. Μπορείτε να μετονομάσετε μια εργασία οποιαδήποτε στιγμή κάνοντας κλικ στο όνομά της στη στήλη **Τίτλος** του πίνακα Ιστορικό Μεταγραφών, επεξεργαζόμενοι το κείμενο και πατώντας `Enter` ή κάνοντας κλικ αλλού.

## Συμπεριφορά Ουράς

- Εάν δεν εκτελείται αυτή τη στιγμή καμία εργασία, το πρώτο αρχείο ξεκινά αμέσως και τα υπόλοιπα εμφανίζονται ως `queued`.
- Εάν μια εργασία εκτελείται ήδη, όλα τα νέα αρχεία που προστέθηκαν εμφανίζονται ως `queued` και θα ξεκινήσουν αυτόματα διαδοχικά.
- Για να παρακολουθήσετε την ενεργή εργασία, κάντε κλικ στο `Monitor` στη στήλη **Ενέργειες** της. Δείτε [Παρακολούθηση Εργασιών](monitoring_jobs.md).
- Για να διακόψετε ή να αφαιρέσετε μια εργασία από την ουρά πριν ξεκινήσει, χρησιμοποιήστε τα κουμπιά `Pause` ή `Remove` στη στήλη **Ενέργειες** της. Δείτε [Παύση, Συνέχιση ή Αφαίρεση Εργασιών](pausing_resuming_removing.md).
