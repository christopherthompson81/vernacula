# Επεξεργασία Μεταγραφών

Ο **Επεξεργαστής Μεταγραφών** σάς επιτρέπει να ελέγχετε την έξοδο ASR, να διορθώνετε κείμενο, να μετονομάζετε ομιλητές απευθείας στη διεπαφή, να προσαρμόζετε τη χρονική τοποθέτηση τμημάτων και να τα επισημαίνετε ως επαληθευμένα — όλα αυτά ενώ ακούτε τον αρχικό ήχο.

## Άνοιγμα του Επεξεργαστή

1. Φορτώστε μια ολοκληρωμένη εργασία (βλ. [Φόρτωση Ολοκληρωμένων Εργασιών](loading_completed_jobs.md)).
2. Στην προβολή **Αποτελέσματα**, κάντε κλικ στο `Edit Transcript`.

Ο επεξεργαστής ανοίγει σε ξεχωριστό παράθυρο και μπορεί να παραμένει ανοιχτός παράλληλα με την κύρια εφαρμογή.

## Διάταξη

Κάθε τμήμα εμφανίζεται ως κάρτα με δύο παράλληλα πλαίσια:

- **Αριστερό πλαίσιο** — η αρχική έξοδος ASR με χρωματική κωδικοποίηση αξιοπιστίας ανά λέξη. Οι λέξεις για τις οποίες το μοντέλο ήταν λιγότερο βέβαιο εμφανίζονται με κόκκινο χρώμα· οι λέξεις υψηλής αξιοπιστίας εμφανίζονται με το κανονικό χρώμα κειμένου.
- **Δεξί πλαίσιο** — ένα επεξεργάσιμο πλαίσιο κειμένου. Κάντε εδώ τις διορθώσεις σας· η διαφορά σε σχέση με το πρωτότυπο επισημαίνεται καθώς πληκτρολογείτε.

Η ετικέτα ομιλητή και το χρονικό εύρος εμφανίζονται πάνω από κάθε κάρτα. Κάντε κλικ σε μια κάρτα για να την εστιάσετε και να αποκαλυφθούν τα εικονίδια ενεργειών της. Τοποθετήστε τον δείκτη του ποντικιού πάνω από οποιοδήποτε εικονίδιο για να δείτε ένα tooltip που περιγράφει τη λειτουργία του.

## Υπόμνημα Εικονιδίων

### Μπάρα Αναπαραγωγής

| Εικονίδιο | Ενέργεια |
|-----------|----------|
| ▶ | Αναπαραγωγή |
| ⏸ | Παύση |
| ⏮ | Μετάβαση στο προηγούμενο τμήμα |
| ⏭ | Μετάβαση στο επόμενο τμήμα |

### Ενέργειες Κάρτας Τμήματος

| Εικονίδιο | Ενέργεια |
|-----------|----------|
| <mdl2 ch="E77B"/> | Ανάθεση του τμήματος σε διαφορετικό ομιλητή |
| <mdl2 ch="E916"/> | Ρύθμιση χρόνων έναρξης και λήξης τμήματος |
| <mdl2 ch="EA39"/> | Απόκρυψη ή επαναφορά απόκρυψης του τμήματος |
| <mdl2 ch="E72B"/> | Συγχώνευση με το προηγούμενο τμήμα |
| <mdl2 ch="E72A"/> | Συγχώνευση με το επόμενο τμήμα |
| <mdl2 ch="E8C6"/> | Διαίρεση του τμήματος |
| <mdl2 ch="E72C"/> | Εκ νέου εκτέλεση ASR σε αυτό το τμήμα |

## Αναπαραγωγή Ήχου

Μια μπάρα αναπαραγωγής εκτείνεται στο πάνω μέρος του παραθύρου του επεξεργαστή:

| Στοιχείο ελέγχου | Ενέργεια |
|------------------|----------|
| Εικονίδιο Αναπαραγωγής / Παύσης | Έναρξη ή παύση αναπαραγωγής |
| Μπάρα αναζήτησης | Σύρετε για μετάβαση σε οποιαδήποτε θέση στον ήχο |
| Ρυθμιστικό ταχύτητας | Ρύθμιση ταχύτητας αναπαραγωγής (0,5× – 2×) |
| Εικονίδια Προηγούμενου / Επόμενου | Μετάβαση στο προηγούμενο ή επόμενο τμήμα |
| Αναπτυσσόμενο μενού λειτουργίας αναπαραγωγής | Επιλογή μιας από τις τρεις λειτουργίες αναπαραγωγής (βλ. παρακάτω) |
| Ρυθμιστικό έντασης | Ρύθμιση έντασης αναπαραγωγής |

Κατά την αναπαραγωγή, η λέξη που εκφωνείται εκείνη τη στιγμή επισημαίνεται στο αριστερό πλαίσιο. Όταν η αναπαραγωγή τεθεί σε παύση μετά από αναζήτηση, η επισήμανση ενημερώνεται στη λέξη που αντιστοιχεί στη θέση αναζήτησης.

### Λειτουργίες Αναπαραγωγής

| Λειτουργία | Συμπεριφορά |
|------------|-------------|
| `Single` | Αναπαραγωγή του τρέχοντος τμήματος μία φορά και στη συνέχεια διακοπή. |
| `Auto-advance` | Αναπαραγωγή του τρέχοντος τμήματος· όταν ολοκληρωθεί, επισημαίνεται ως επαληθευμένο και γίνεται μετάβαση στο επόμενο. |
| `Continuous` | Αναπαραγωγή όλων των τμημάτων διαδοχικά χωρίς να επισημανθεί κανένο ως επαληθευμένο. |

Επιλέξτε την ενεργή λειτουργία από το αναπτυσσόμενο μενού στη μπάρα αναπαραγωγής.

## Επεξεργασία Τμήματος

1. Κάντε κλικ σε μια κάρτα για να την εστιάσετε.
2. Επεξεργαστείτε το κείμενο στο δεξί πλαίσιο. Οι αλλαγές αποθηκεύονται αυτόματα όταν μετακινείτε την εστίαση σε άλλη κάρτα.

## Μετονομασία Ομιλητή

Κάντε κλικ στην ετικέτα ομιλητή μέσα στην εστιασμένη κάρτα και πληκτρολογήστε ένα νέο όνομα. Πατήστε `Enter` ή κάντε κλικ αλλού για αποθήκευση. Το νέο όνομα εφαρμόζεται μόνο σε αυτή την κάρτα· για να μετονομάσετε έναν ομιλητή καθολικά, χρησιμοποιήστε την επιλογή [Επεξεργασία Ονομάτων Ομιλητών](editing_speaker_names.md) από την προβολή Αποτελέσματα.

## Επαλήθευση Τμήματος

Κάντε κλικ στο πλαίσιο ελέγχου `Verified` σε μια εστιασμένη κάρτα για να την επισημάνετε ως ελεγμένη. Η κατάσταση επαλήθευσης αποθηκεύεται στη βάση δεδομένων και είναι ορατή στον επεξεργαστή σε μελλοντικές φορτώσεις.

## Απόκρυψη Τμήματος

Κάντε κλικ στο `Suppress` σε μια εστιασμένη κάρτα για να αποκρύψετε το τμήμα από τις εξαγωγές (χρήσιμο για θόρυβο, μουσική ή άλλες ενότητες μη ομιλίας). Κάντε κλικ στο `Unsuppress` για να το επαναφέρετε.

## Ρύθμιση Χρόνων Τμήματος

Κάντε κλικ στο `Adjust Times` σε μια εστιασμένη κάρτα για να ανοίξετε το παράθυρο διαλόγου ρύθμισης χρόνων. Χρησιμοποιήστε τον τροχό κύλισης πάνω από το πεδίο **Έναρξη** ή **Λήξη** για να μετακινήσετε την τιμή σε βήματα 0,1 δευτερολέπτου, ή πληκτρολογήστε απευθείας μια τιμή. Κάντε κλικ στο `Save` για εφαρμογή.

## Συγχώνευση Τμημάτων

- Κάντε κλικ στο `⟵ Merge` για να συγχωνεύσετε το εστιασμένο τμήμα με το αμέσως προηγούμενο.
- Κάντε κλικ στο `Merge ⟶` για να συγχωνεύσετε το εστιασμένο τμήμα με το αμέσως επόμενο.

Το συνδυασμένο κείμενο και το χρονικό εύρος και των δύο καρτών ενώνονται. Αυτό είναι χρήσιμο όταν μια ενιαία εκφωνημένη φράση έχει διαιρεθεί σε δύο τμήματα.

## Διαίρεση Τμήματος

Κάντε κλικ στο `Split…` σε μια εστιασμένη κάρτα για να ανοίξετε το παράθυρο διαλόγου διαίρεσης. Τοποθετήστε το σημείο διαίρεσης μέσα στο κείμενο και επιβεβαιώστε. Δημιουργούνται δύο νέα τμήματα που καλύπτουν το αρχικό χρονικό εύρος. Αυτό είναι χρήσιμο όταν δύο διακριτές εκφωνήσεις έχουν συγχωνευθεί σε ένα τμήμα.

## Εκ Νέου Εκτέλεση ASR

Κάντε κλικ στο `Redo ASR` σε μια εστιασμένη κάρτα για να εκτελέσετε εκ νέου την αναγνώριση ομιλίας στον ήχο αυτού του τμήματος. Το μοντέλο επεξεργάζεται μόνο το τμήμα ήχου για εκείνο το τμήμα και παράγει μια νέα μεταγραφή από μία μοναδική πηγή.

Χρησιμοποιήστε αυτή τη λειτουργία όταν:

- Ένα τμήμα προέκυψε από συγχώνευση και δεν μπορεί να διαιρεθεί (τα συγχωνευμένα τμήματα καλύπτουν πολλαπλές πηγές ASR· η εκ νέου εκτέλεση ASR τα συμπτύσσει σε ένα, μετά από το οποίο το `Split…` γίνεται διαθέσιμο).
- Η αρχική μεταγραφή είναι κακής ποιότητας και επιθυμείτε μια καθαρή δεύτερη προσπάθεια χωρίς χειροκίνητη επεξεργασία.

**Σημείωση:** Οποιοδήποτε κείμενο έχετε ήδη πληκτρολογήσει στο δεξί πλαίσιο απορρίπτεται και αντικαθίσταται με τη νέα έξοδο ASR. Η λειτουργία απαιτεί το αρχείο ήχου να είναι φορτωμένο· το κουμπί απενεργοποιείται αν ο ήχος δεν είναι διαθέσιμος.
