# Λήψη Μοντέλων

Το Vernacula-Desktop απαιτεί αρχεία μοντέλων AI για να λειτουργήσει. Αυτά δεν συμπεριλαμβάνονται στην εφαρμογή και πρέπει να ληφθούν πριν από την πρώτη σας μεταγραφή.

## Κατάσταση Μοντέλων (Αρχική Οθόνη)

Μια λεπτή γραμμή κατάστασης στην κορυφή της αρχικής οθόνης δείχνει εάν τα μοντέλα σας είναι έτοιμα. Όταν λείπουν αρχεία, εμφανίζεται επίσης ένα κουμπί `Open Settings` που σας οδηγεί απευθείας στη διαχείριση μοντέλων.

| Κατάσταση | Σημασία |
|---|---|
| `All N model file(s) present ✓` | Όλα τα απαιτούμενα αρχεία έχουν ληφθεί και είναι έτοιμα. |
| `N model file(s) missing: …` | Ένα ή περισσότερα αρχεία απουσιάζουν· ανοίξτε τις Ρυθμίσεις για λήψη. |

Όταν τα μοντέλα είναι έτοιμα, τα κουμπιά `New Transcription` και `Bulk Add Jobs` ενεργοποιούνται.

## Πώς να Κατεβάσετε Μοντέλα

1. Στην αρχική οθόνη, κάντε κλικ στο `Open Settings` (ή μεταβείτε στο `Settings… > Models`).
2. Στην ενότητα **Models**, κάντε κλικ στο `Download Missing Models`.
3. Εμφανίζεται μια γραμμή προόδου και μια γραμμή κατάστασης που δείχνουν το τρέχον αρχείο, τη θέση του στην ουρά και το μέγεθος λήψης — για παράδειγμα: `[1/3] encoder-model.onnx — 42 MB`.
4. Περιμένετε μέχρι η κατάσταση να εμφανίσει `Download complete.`

## Ακύρωση Λήψης

Για να διακόψετε μια λήψη που βρίσκεται σε εξέλιξη, κάντε κλικ στο `Cancel`. Η γραμμή κατάστασης θα εμφανίσει `Download cancelled.` Τα αρχεία που έχουν ληφθεί εν μέρει διατηρούνται, ώστε η λήψη να συνεχιστεί από εκεί που σταμάτησε την επόμενη φορά που θα κάνετε κλικ στο `Download Missing Models`.

## Σφάλματα Λήψης

Εάν μια λήψη αποτύχει, η γραμμή κατάστασης εμφανίζει `Download failed: <reason>`. Ελέγξτε τη σύνδεσή σας στο διαδίκτυο και κάντε ξανά κλικ στο `Download Missing Models` για επανάληψη. Η εφαρμογή συνεχίζει από το τελευταίο αρχείο που ολοκληρώθηκε με επιτυχία.

## Αλλαγή Ακρίβειας

Τα αρχεία μοντέλων που πρέπει να ληφθούν εξαρτώνται από την επιλεγμένη **Model Precision**. Για να την αλλάξετε, μεταβείτε στο `Settings… > Models > Model Precision`. Εάν αλλάξετε ακρίβεια μετά τη λήψη, το νέο σύνολο αρχείων πρέπει να ληφθεί ξεχωριστά. Δείτε [Επιλογή Ακρίβειας Βαρών Μοντέλου](model_precision.md).
